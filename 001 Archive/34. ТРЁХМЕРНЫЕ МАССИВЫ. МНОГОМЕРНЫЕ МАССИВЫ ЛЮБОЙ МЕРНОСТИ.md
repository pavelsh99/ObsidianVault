> Многомерные массивы бывают как прямоугольные так и зубчатые.

![[Pasted image 20250525100245.png]]

Объявление трехмерного массива
```csharp
int[,,] myArray = new int[4, 3, 5];
```

У тебя есть **тетрадь** с:

- **4 листами** — это первая размерность `[4]`
    
- На **каждом листе** — **3 строки** — вторая размерность `[3]`
    
- В **каждой строке** — **5 клеточек (столбцов)** — третья размерность `[5]`

1 лист Можно представить так 
```csharp
строка 0: [0][0][0][0][0]  
строка 1: [0][0][0][0][0]  
строка 2: [0][0][0][0][0]  
```

>Чем больше индексов при создании массива, тем больше измерений.

Инициализация трехмерного массива вручную:

```csharp
            int[,,] myArray =
            {
                {
                    {1,5,5},
                    {51,37,3}
                }, 
                {
                    {65,24,12},
                    {53,52,84}
                }, 
                {
                    {30,44,26},
                    {30,145,11}
                }
            };
```

Инициализация трехмерного массива перебором `for`

```csharp
            int[,,] myArray = new int [4,3,5];

            Random rnd = new Random();

            for (int i = 0; i < myArray.GetLength(0); i++)
            {
                for (int j = 0; j < myArray.GetLength(1); j++)
                {
                    for (int k = 0; k < myArray.GetLength(2); k++)
                    {
                        myArray[i, j, k] = rnd.Next(100);
                    }
                }
            }
```

Вывод на консоль:

```csharp
            for (int i = 0; i < myArray.GetLength(0); i++)
            {
                Console.WriteLine("page " + (i+1) + ":");
                for (int j = 0; j < myArray.GetLength(1); j++)
                {
                    for (int k = 0; k < myArray.GetLength(2); k++)
                    {
                        Console.Write(myArray[i, j, k] + "\t");
                    }
                    Console.WriteLine();
                }
                Console.WriteLine("\n");
            }
```

Зубчатый массив , создание и инициализация:

```csharp
        Random rnd = new Random();

        // Создаем зубчатый 3D массив
        int[][][] jaggedArray = new int[2][][];

        for (int i = 0; i < jaggedArray.Length; i++)
        {
            // Кол-во "полок" в каждом "шкафу"
            jaggedArray[i] = new int[rnd.Next(1, 4)][];

            for (int j = 0; j < jaggedArray[i].Length; j++)
            {
                // Кол-во "книг" на каждой "полке"
                jaggedArray[i][j] = new int[rnd.Next(1, 5)];

                for (int k = 0; k < jaggedArray[i][j].Length; k++)
                {
                    jaggedArray[i][j][k] = rnd.Next(100); // Заполняем случайным числом
                }
            }
        }
```

Вывод зубчатого массива на консоль:

```csharp
   // Вывод массива
        for (int i = 0; i < jaggedArray.Length; i++)
        {
            Console.WriteLine($"Шкаф {i}:");
            for (int j = 0; j < jaggedArray[i].Length; j++)
            {
                Console.Write($"  Полка {j}: ");
                for (int k = 0; k < jaggedArray[i][j].Length; k++)
                {
                    Console.Write(jaggedArray[i][j][k] + " ");
                }
                Console.WriteLine();
            }
            Console.WriteLine();
        }
```
