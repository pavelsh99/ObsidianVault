>Инициализация массива - это его объявление и выделение для него места в оперативной памяти.

```csharp
int[] myArray = new int[5];  // такая инициализация даст пять нулей
```
>`[5]` Это явное количество элементов

#### Существуют разные способы инициализации

##### 1 Способ

```csharp
int[] myArray = new int[5] {4, 5, 6, 7, 8};
```

Мы объявляем массив и через запятую указываем наши элементы.

>Если мы укажем меньше элементов чем указанно, то появится ошибка

##### 2 Способ

```csharp
int[] myArray = new int[] {4, 7, 8};// не указываем сколько элементов в []
```

Так создастся массив с количеством элементов, которое мы указали в скобках. ( тут 3)

>Таким образом можно избежать ошибок.
>Но если мы ничего не указали в квадратных скобках, в фигурных обязательно надо что-то указать!

При такой записи можно не указывать тип элемента после `new`.

```csharp
int[] myArray = new [] {4, 7, 8};
```

Можно еще упростить

```csharp
int[] myArray = {4, 7, 8}
```
##### 3 Способ

Если массив нужно инициализировать определенными значениями, но не 0:

Используем статический класс `Enumerable` с методом `Rrepeat` и метод `ToArray()`

Он повторяет какой то элемент определенное количество раз

```csharp
int[] myArray = Enumerable.Repeat(5,10).ToArray(); 

//5 это элемент 10 это сколько элементов
```

Можно использовать статический класс `Enumerable` с методом `Range` и метод `ToArray()`

Он перечислит элементы, начиная с заданного, определенное количество раз

```csharp
int[] myArray = Enumerable.Range(4,5).ToArray();

//(4 - это первое число 5 - это сколько шагов)
```
