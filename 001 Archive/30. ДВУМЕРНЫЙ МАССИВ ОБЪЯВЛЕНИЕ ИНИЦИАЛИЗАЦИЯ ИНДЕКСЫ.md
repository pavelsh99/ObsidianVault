# Объявление двумерного массива

```csharp
тип_данных [,] имя_массива;// отличие - запятая [,]
```

От запятой зависит какое количество измерений будет в массиве.

```csharp
int[,] myArray; //так объявляется двумерный массив
```

чтобы  выделить место в оперативной памяти мы должны решить сколько строк и столбцов он будет иметь. У нас будет 3 строки и 5 столбцов.

```csharp
            int[,] myArray;

            myArray = new int[3,5];
```

# Инициализация двумерного массива

Так мы можем внести переменные в массив по советующему индексу.
```csharp
int[,] myArray = new int[3, 5];//в одну строку

myArray[0,2] = 99;
Console.WriteLine(myArray[0,2]);
```
Так мы сразу можем заполнить массив представив его как таблицу с 3 строками и 5 столбцами:
	шаг 1
```csharp
            int[,] myArray = new int[3, 5]//<- тут нет ;
            {
                { },
                { },
                { }
            };
```

	шаг 2
```csharp
            int[,] myArray = new int[3, 5]
            {
                {12,51,43,12,14},
                {44,24,63,18,76},
                {15,98,22,17,54}
            }; 
```

>Но если мы как тут пишем явно 3 строки и 5 колонок, то нужно будет именно столько указывать переменных.

	можно так 
```csharp
            int[,] myArray =
            {
                {12,51,43,12,14},
                {44,24,63,18,76},
                {15,98,22,17,54}
            }; 
```

	или так
```csharp
            int[,] myArray = new int[,]
            {
                {12,51,43,12,14},
                {44,24,63,18,76},
                {15,98,22,17,54}
            }; 
```
