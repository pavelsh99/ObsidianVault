Ключевое слово `continue` позволяет прервать выполнение какой либо итерации цикла но не выходить из него полностью.

```csharp
            for (int i = 0; i < 5 ; i++)
            { 

                if (i == 2)
                {
                    continue;
                }
                Console.WriteLine(i);
            }
```

Консоль выведет следующее: ![[Pasted image 20250515194906.png]]

То есть, когда `i = 2` случилось `continue` и программа не выполняла `console.WriteLine(i)` а начала следующую итерацию.
