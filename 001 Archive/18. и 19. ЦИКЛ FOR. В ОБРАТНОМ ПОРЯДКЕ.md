# Цикл `For`

[# C# ЦИКЛ FOR | C# ОТ НОВИЧКА К ПРОФЕССИОНАЛУ | # 18](https://www.youtube.com/watch?v=Y2SX6PykpxA&list=PLQOaTSbfxUtD6kMmAYc8Fooqya3pjLs1N&index=25)

Цикл `for` повторяет  код несколько раз как и `while` но имеет особенности, например встроенный оператор или счетчик:
```csharp
for (int i = 0; i < 10; i++)
{  
    Console.WriteLine(i);  
}
```

>`i`- Это итератор или счетчик

Простыми словами, этот `for`-цикл можно описать так:

> "Начни с переменной `i`, которая равна 0. Если `i` меньше 10 — выводи ее на консоль. После каждой итерации увеличивай `i` на 1."

Он состоит из трех действий
	1)Объявление переменной
	2)Прибавление 1
	3)Проверка условия (`i < 10`)
		4) Вывод на консоль


`For` С выводом на консоль :

```csharp
// Цикл for  
int limit = int.Parse(Console.ReadLine());  for (int i = 0; i < limit; i++)  
{  
    Console.WriteLine(i);  
}
```

Если делать то же самое но с `while`, то будет так : 

```csharp
int limit = int.Parse(Console.ReadLine()); int i = 0;  
while (i < limit) ;  
{  
    i++;    
    Console.WriteLine(i);  
}
```

### Отличие от `while`

Отличается область видимости: Когда мы объявляем `i` вне `while`, то мы имеем к ней доступ везде, и мы можем ее менять.

```csharp
int i = 1000;  
while (i < 10) ;  
{  
    i++;    Console.WriteLine(i);  
}
```

То есть переменная может помешать в большом коде.

А в цикле `for` она не будет видна негде, что удобно в большом проекте.

# Цикл `for` в обратном порядке | с несколькими условиями | с несколькими переменными

- Используется при работе с несколькими переменными в цикле `for`
- Работа с циклом `for` в обратном порядке, например чтобы перебрать элементы массива в обратном порядке.
- Несколько условий с циклом `for`

>В `for` можно объявить несколько переменных действий над ними.

```csharp
int limit = int.Parse(Console.ReadLine());  
for (int i = 0, j = 2; i < limit; i++, j++)  
{  
    Console.WriteLine(j);  
    Console.WriteLine(i);  
}
```

>Можно оставлять секции `for`пустыми : 

```csharp
for (;;)  

{  
Console.WriteLine(i); 
}
```

>Можно добавить несколько условий с помощью [[14. ЛОГИЧЕСКИЕ ОПЕРАТОРЫ. СОКРАЩЁННЫЕ ЛОГИЧЕСКИЕ ОПЕРАЦИИ|логических операторов]] : 

```csharp
int limit = int.Parse(Console.ReadLine());  
for (int i = 0, j = 2; i < 10 %% j < 12; i++, j++)  
{  
    Console.WriteLine(j);  
    Console.WriteLine(i);  
}
```
