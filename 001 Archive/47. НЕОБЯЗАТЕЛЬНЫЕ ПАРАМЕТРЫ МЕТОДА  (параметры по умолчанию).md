**Необязательный параметр** — это параметр метода, которому можно заранее задать **значение по умолчанию**.  

Если при вызове метода этот параметр не указан, используется это значение.

- Необязательные параметры всегда должны идти **после обязательных**.

- Можно указать несколько необязательных параметров подряд.

### Пример

```csharp
static int Sum(int a, int b, bool enablelogging = false)
{
    int result = a + b;
    
    if (enablelogging)
    {
        Console.WriteLine("Значение переменной a = " + a);
        Console.WriteLine("Значение переменной b = " + b);
        Console.WriteLine("Результат сложения = " + result);
    }
    
    return result;
}
```

### Вызов метода

```csharp
int result1 = Sum(1, 5); 
// параметр enablelogging не указан → по умолчанию false

int result2 = Sum(3, 7, true); 
// параметр enablelogging указан → true
```

### Вывод при включенном логе

```csharp
Значение переменной a = 3
Значение переменной b = 7
Результат сложения = 10
```

### Главное для конспекта:

1. Необязательные параметры позволяют **не перегружать метод** лишними вариантами. 

Вместо:
```csharp
Sum(int a, int b) 
Sum(int a, int b, bool enablelogging)
```
достаточно одного метода.

2. Если аргумент не указан → берётся **значение по умолчанию**.

3. Указывать нужно **после обязательных параметров**.
