**Одномерный массив** - это совокупность элементов одного типа или коллекция данных.

Массивы содержат данные **одного типа**.

Каждый элемент объединен **одним и тем же именем**, что удобно

Вызывать данные можно, обращаясь к **имении** массива и **индексу** переменной внутри него.

Массив это как многоквартирный дом, Адрес дома это имя массива, а номер квартиры это индекс переменной.

Данные массива лежат в оперативной памяти.

Массивы удобны для большого количества переменных или если не понятно сколько должно быть переменных.

Массив объявляется так :
```csharp
int[] myArray;// myArray - это имя массива.
```

Далее необходимо выделить место в оперативной  памяти (hip) памяти для массива:

```csharp
myArray = new int[5];
```

Мы указываем 
 - имя массива
 - потом оператор `=` 
 - затем ключевое слово `new`
 - тип элементов (`int` в данном случае) 
 - их количество (тут 5).

так выглядит оперативная память , которую мы выделили для пяти `int`:

>ПОРЯДОК ЭЛЕМЕНТОВ НАЧИНАЕТСЯ С НУЛЯ. 

Так можно присвоить переменной с индексом 1 значение.

```csharp
myArray = new int[5];
myArray[1] = 3;
```

Значение второй переменной теперь 3

![[Pasted image 20250518204609.png]]

Так мы можем обратиться к этой переменной:

```csharp
Console.WriteLine(myArray[1]);
```

Можно извлечь данные в отдельную переменную.

```csharp
myArray = new int[5];  
  
myArray[1] = 3;  
  
int a = myArray[1];//тут  
  
Console.WriteLine(myArray[1]);
```

>Если мы попытаемся выйти  за границы массива мы получим исключение (например если обратимся к переменной, индекса которой не существует)
