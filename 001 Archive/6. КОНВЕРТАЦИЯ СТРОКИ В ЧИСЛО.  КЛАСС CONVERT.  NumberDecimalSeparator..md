### Конвертировать строку в числа можно используя класс Convert
```csharp
        static void Main(string[] args)
        {
            string str;

            int a, b, result;

            Console.WriteLine("Напишите число 1");

            str = Console.ReadLine();

            a = Convert.ToInt32(str);

            Console.WriteLine("Напишите число 2");

            str = Console.ReadLine();

            b = Convert.ToInt32(str);

            result = (a + b);

            Console.WriteLine("Результат сложения = "+ result);
        }
```

Чтобы избежать проблем с запятой/точкой  при конвертации в строки `double`, подключается пространство имен `System.Globalisation`, где есть класс `NumberFormatInfo`. В нем можно указать что является разделителем, точка или запятая.
```Csharp
using System;
using System.Globalization;

namespace Lesson2
{
    internal class Program
    {
        static void Main(string[] args)
        {
            string str = "1.2";

            NumberFormatInfo numberFormatInfo = new NumberFormatInfo()
            {
                NumberDecimalSeparator = "."
            };

            double a = Convert.ToDouble(str, numberFormatInfo);
            Console.WriteLine(a); // Выведет: 1.2
        }
    }
}
```
