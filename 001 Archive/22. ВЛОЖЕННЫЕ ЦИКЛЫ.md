Чаще всего вложенные циклы используются при работе с массивами, где чтобы перебрать все элементы массива, это удобно.

Также они используются при работе с данными в ООП, если эти данные находятся в структуре: объект, внутри него другие объекты, внутри них другие объекты. И нужно что- то сделать каждым вложенным объектом и иметь к нему доступ.

Или при сортировке массивов.

>Вложенным циклом, считается цикл внутри другого цикла

```csharp
            while (true) //Цикл while
            {
                for(int i = 0; i < 100;i++)//в него вложен for
                {

                }
            }
```

Пример:
```csharp 
            for(int i = 1; i <= 3;  i++)
            {
                Console.WriteLine("Итерация номер " + i);
                for (int j = 1; j < 5; j++)
                {
                    Console.WriteLine("Цикл 2, итерация номер " + j);
                }
            }
            Console.ReadLine();
```

Таким образом в одной итерации первого цикла 4 итерации вложенного цикла, который исполняется 3 раза. 

Программа, которая рисует прямоугольник указанного размера:

```csharp
           Console.Write("Введите Высоту:");
           int height = int.Parse(Console.ReadLine());

           Console.Write("Введите Ширину:");
           int width = int.Parse(Console.ReadLine());


           for (int j  = 0; j < height; j++)
           {
               for (int i = 0; i < width; i++)
               {
                   Console.Write("#");
               }
               Console.WriteLine();
           }

           Console.ReadLine();
```
