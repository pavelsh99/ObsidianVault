### Почему в C# нет функций, но есть методы

>Функции и методы это практически одно и то же но функций в C# не существует.

>Функция, которая является членом класса - это метод. А в C# функции без класса быть не может. Поэтому есть только методы

### Всё что относится к методам, относится и к методам класса

>Это модификаторы, ref, out, in, params и т.д.

### Пример написания метода класса.

>Если мы напишем метод класса внутри него,  мы сможем вызывать его подобно `Console.WriteLine();` , то есть использовать его на уровне объекта класса.

```csharp
    internal class Student
    {
        public Guid id;
        public string firstName;
        public string lastName;
        public int age;
        public string group;

        public void Print()
        {
            Console.WriteLine("Информация о студенте: ");
            Console.WriteLine($"Имя - {firstName}");
            Console.WriteLine($"Фамилия - {lastName}");
            Console.WriteLine($"Id - {id}");
            Console.WriteLine($"Возраст - {age}");
            Console.WriteLine($"Группа - {group}");
        }
    }
```

>Особенность: мы обращаемся к полям класса( `age` `name` и т.д. ) напрямую, а не через объект класса ( не `student.id`, а просто `id`) 

```csharp
        static void Main(string[] args)
        {
            var student1 = GetStudent();
            student1.Print();
        }
```

>Теперь мы можем вызвать метод `Print` у *конкретного* экземпляра класса `student1`

### У каждого объекта своя информация

>Если мы создадим  объект`student2` и у него вызовем  `print`, то у него не будет информации.

```csharp
        static void Main(string[] args)
        {
            var student1 = GetStudent();
            student1.Print();
```

### Пример метода с возвращаемым значением.

>Допишем в класс метод, который будет возвращать имя студента.

```csharp
    internal class Student
    {
        public Guid id;
        public string firstName;
        public string lastName;
        public int age;
        public string group;

        public void Print()
        {
            Console.WriteLine("Информация о студенте: ");
            Console.WriteLine($"Имя - {firstName}");
            Console.WriteLine($"Фамилия - {lastName}");
            Console.WriteLine($"Id - {id}");
            Console.WriteLine($"Возраст - {age}");
            Console.WriteLine($"Группа - {group}");
        }
        public string GetFullName()  // ТУТ
        {
            return $"{lastName} {firstName}";
        }
```

>Теперь вызовем этот метод через объект класса.

```csharp
    Console.WriteLine(student1.GetFullName());// вывод: Босков Николай
```
