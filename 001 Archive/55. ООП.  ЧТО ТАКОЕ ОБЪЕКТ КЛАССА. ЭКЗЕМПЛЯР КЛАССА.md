### Что такое ООП?

>ООП это способ облегчения восприятия кода для человека при помощи классов и структур.

### Принципы ООП

>Инкапсуляция
>Наследование
>Полиморфизм

- Инкапсуляция это сокрытие внутренних элементов объекта класса и оставление только внешних функций

- Наследование это принцип, позволяющий одному классу (дочернему или производному) получить свойства и методы другого класса (родительского или базового).

- Полиморфизм это свойство, позволяющее обрабатывать данные разных типов одним образом.


### Практика

>Допустим, у нас есть точка в пространстве.

>Класс это по сути наш тип данных

>Класс это как бы чертеж, но объект класса это уже готовый дом

        class Point

>Создадим объект класса

##### Синтаксис

>`<Класс(тип данных)> <имя объекта> = new <Класс>();`

```csharp
Point p = new Point();
```

> Класс это ссылочный тип. Он хранится в управляемой куче и  с помощью`new` мы выделяем там место. Иначе объект `== null`

Поместим данные о  первой точке в поля.
```csharp
            Point p = new Point();
            p.x = 4;
            p.y = 2;
```

- Теперь мы можем использовать поля и обращаться к ним
```csharp
            Console.WriteLine(p.x);
            Console.WriteLine(p.y);
```

### Экземпляры класса не связаны

```csharp
            Point p2 = new Point();
            p2.x = 2;
            p2.y = 6;

            Point p = new Point();
            p.x = 4;
            p.y = 2;
```

### Класс - это ссылочный тип.

```csharp
            Point p2 = new Point();
            p2 = null;
            p2.x = 2;// == 0
            p2.y = 6;// == 0
```

>По сути выполнился код только на первой строчке потому что ссылка была разорвана, поэтому было присвоено значение по умолчанию

### Описываем студента

```csharp
class Student // Класс студент
{
    public Guid id;
    public string firstName;
    public string lastName;
    public int age;
    public string group;
}

static Student GetStudent() 
//                         Метод возвращает студента и его данные
{
    var student = new Student();

    student.firstName = "Николай";
    student.lastName = "Босков";
    student.id = Guid.NewGuid();
    student.age = 20;
    student.group = "ПМД_23";

    return student;
}

static void PrintStudent(Student student) // Местод выводит данные о студенте
{
    Console.WriteLine("Информация о студенте: ");
    Console.WriteLine($"Имя - {student.firstName}");
    Console.WriteLine($"Фамилия - {student.lastName}");
    Console.WriteLine($"Id - {student.id}");
    Console.WriteLine($"Возраст - {student.age}");
    Console.WriteLine($"Группа - {student.group}");
}
static void Main(string[] args)
{
    var student = GetStudent(); // обращение к методам
    PrintStudent(student);
}
```
