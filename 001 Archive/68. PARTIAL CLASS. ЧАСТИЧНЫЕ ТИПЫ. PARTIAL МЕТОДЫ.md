# Что такое `partial`?

Слово `partial` позволяет разнести части обычного класса по разным файлам.

При этом в работе с таким классом мы не заметим разницы(например при создании объекта класса или при вызове метода)

# Когда использовать `partial`?

> Это редкая вещь и просто так разбивать класс на части смысла нет.

> Даже если класс большой

> При этом может быть так, что есть часть класса, куда лучше не лезть, а есть часть которую нужно изменить особым образом.

# Синтаксис `partial`

> Пример `partial` класса:
```csharp
    partial class Person
    {
        public string FirstName { get; set; }
        public string LastName { get; set; }

        public Person(string firstName, string lastName)
        {
            FirstName = firstName; 
            LastName = lastName;
        }

        public string GetFullName()
        {
            return (FirstName + " " + LastName);
        }
        
        public void PrintFullName()
        {
            Console.WriteLine(GetFullName());
        }
    }
```

> Чтобы класс стал `partial` добавляем ключевое слово перед `class`
```csharp
partial class Person
{
	
}
```

# `partial` в WinForms

> `partial` классы активно применяются в винформс. 

> Когда мы добавляем компоненты в дизайнере формы автоматически добавляется код под наши компоненты
