- Операторы это синтаксические конструкции, которые позволяют совершать операции над данными, давая результат.
- Данные над которыми производится операции, называются ***операнды***
- Операторы делятся в зависимости от количества операнд на:
	- Унарные(1 операнд)
	- Бинарные(2 операнда)
	- Тернарные (3 операнда)
Арифметические операторы:
1. Сложение `+`
2. Вычитание `-`
3. Умножение `*`
4. Деление `/`
5. Деление по модулю/деление с остатком `%` (дает остаток по делению)( с помощью него можно проверить кратность делению, четность или нечетность числа)
6. Сокращенные
		| <span style="color:#e06c75;">+=</span> | <span style="color:#61afef;">x = x + y</span> | Прибавить |
		| <span style="color:#e06c75;">-=</span> | <span style="color:#61afef;">x = x - y</span>   | Вычесть |
		| <span style="color:#e06c75;">*=</span> | <span style="color:#61afef;">x = x * y</span>   | Умножить |
		| <span style="color:#e06c75;">/=</span> | <span style="color:#61afef;">x = x / y</span>   | Разделить |
		| <span style="color:#e06c75;">%=</span> | <span style="color:#61afef;">x = x % y</span> | Остаток |
