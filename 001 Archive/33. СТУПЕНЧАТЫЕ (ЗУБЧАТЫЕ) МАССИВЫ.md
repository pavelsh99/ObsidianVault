> Зубчатые массивы содержат разное количество элементов в строке.

>Зубчатый массив выглядит как одномерный массив, который содержит массивы внутри себя.
>Поэтому зубчатые массивы одномерные
>![[Pasted image 20250524124850.png]]


## Как объявить ступенчатый массив?

```csharp
int[][] myArray = new int [5][];
```

В квадратных скобках указываем сколько элементов будет этой части  массива

![[Pasted image 20250524125429.png]]
 чтобы выделить место во вложенных массивах
```csharp
            int[][] myArray = new int[3][];
            myArray[0] = new int[10];
            myArray[1] = new int[3];
            myArray[2] = new int[5];
```
Ввод длины ступенчатого массива и длин вложенных массивов
```csharp
            Console.WriteLine("\n Enter lenght of Array");
            int temp = int.Parse(Console.ReadLine());
            int[][] myArray = new int[temp][];
            for (int i  = 0; i < myArray.Length; i++)
            {
                Console.WriteLine("\n Enter lenght of Array number " + i);
                myArray[i] = new int[int.Parse(Console.ReadLine())] ;
            }

```

```csharp
            int[][] myArray = new int[3][];
            myArray[0] = new int[10];
            myArray[1] = new int[3];
            myArray[2] = new int[5];
            
            myArray[0][3] = 55;

            Console.WriteLine(myArray[0][3]);// массив инд 0 переменная под инд 3
```

Заполнение массива случайными числами до 900

```csharp
            Random random = new Random();   

            for (int i  = 0; i < myArray.Length; i++)
            {
                for (int j = 0; j < myArray[i].Length; j++)
                {
                    myArray[i][j] = random.Next(900);
                }
            }
```

вывод на консоль массива

```csharp
            for (int i = 0; i < myArray.Length; i++)
            {
                for (int j = 0; j < myArray[i].Length; j++)
                {
                    Console.Write(myArray[i][j] + "\t");
                }
                Console.WriteLine();
            }
```
