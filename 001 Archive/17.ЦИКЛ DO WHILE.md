Цикл `do while` похож на `while`, но работает по-другому:
`while` проверяет истинность условия и совершает действие пока оно истинно.
```csharp
while(true)//Условие
{
Console.Writeline("Выполнить действие")//Действие
}
```

`do while` ==сначала совершает действие,== а только ==потом проверяет условие==.

```csharp
do // 1) Сначала выполняется действие
{

} while{false} //2) проверяется условие даже если false то выполнит 1 раз
```
