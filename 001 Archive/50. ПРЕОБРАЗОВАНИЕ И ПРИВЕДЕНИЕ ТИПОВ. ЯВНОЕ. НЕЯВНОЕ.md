Преобразование (приведение) это процесс конвертации объекта одного типа данных в другой.

>Бывает:
явное (explicit) (явно прописывается)
неявное  (делается автоматически)

Пример явного приведения:

```csharp
double a = 5.5;
float b = (float)a;
```

Пример неявного приведения:

```csharp
int a = 5;
double b = a;
```

### Сужающее и расширяющее преобразование

Сужающее:
```csharp
double a = 5.5;
float b = (float)a;
```

У `float` больше диапазон значений, чем у `double`, поэтому при явном приведении мы сужаем диапазон значений.

- То есть мы потенциально больший объект кладем в меньшую коробку

Расширяющее:
```csharp
int a = 5;
double b = a;
```

у int больший диапазон значений, а мы кладем его в double.
