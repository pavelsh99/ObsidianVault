# Конструкция `if` `else`

Конструкция `if` `else` позволяет осуществить ветвление кода.

Она всегда должна принимать данные в формате `true` `false` 

```csharp
     if (утверждение или выражение) 
     {


    }
    else
    {


    }
```

## Пример утверждения:

```csharp
            bool isinfected = true;

            if (isinfected)
            {
                Console.WriteLine("Персонаж заражен!");
            }
            else
            {
                Console.WriteLine("Персонаж здоров!");
            }
```

Пример выражения:

```csharp
            int input;

            Console.WriteLine("Введите число");

            input = int.Parse(Console.ReadLine());

            if (input == 5)
            {
                Console.WriteLine("число равно пяти!");

            }
            else
            {
                Console.WriteLine("число не равно пяти");
            }
```

в выражении может использоваться любой оператор.
