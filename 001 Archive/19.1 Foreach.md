`foreach` — это специальный цикл в C#, который **перебирает все элементы коллекции** (массивов, списков и т.п.) **один за другим**, **без индексов**. Он упрощает чтение и делает код чище.

### Общий синтаксис:

```csharp
foreach (тип переменная in коллекция)
{
    // действия с переменной
}
```

- `тип` — тип элемента (например, `int`, `string` и т.д.)
    
- `переменная` — имя временной переменной (например, `number`)
    
- `коллекция` — объект, по которому ты проходишь (например, массив или `List<T>`)

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };

foreach (int number in numbers)
{
    Console.WriteLine(number);
}

```
### Как это читается по-человечески:

> «Для каждого `number` в массиве `numbers`, напечатай `number`».

```csharp

        string[] names = { "Аня", "Борис", "Вика", "Глеб" };

        foreach (string name in names)
        {
            Console.WriteLine("Привет, " + name + "!");
        }

```

### Что делает этот код:

1. Есть массив `names`, в котором 4 имени.
    
2. Цикл `foreach` берёт **каждое имя по очереди** и сохраняет его во временную переменную `name`.
    
3. Внутри цикла выводится приветствие с этим именем.
> «Для каждого `name` в массиве `names`, напечатай `Привет, name!`»
