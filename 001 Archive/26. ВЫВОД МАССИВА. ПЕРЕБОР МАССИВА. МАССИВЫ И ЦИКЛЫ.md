>Удобный инструмент взаимодействия с массивом - это цикл, обычно это делают с помощью цикла `for` или `for each`.

Чтобы правильно написать цикл, для перебора всех элементов массива, нужно узнать его длину, используя метод `<имя массива>.Lenght`.

Чтобы перебрать массив нужно по очереди обратиться к каждому элементу массива.

Поэтому цикл должен обратиться к массиву такое число раз, которое соответствует количеству элементов массива. То есть цикл должен проделать столько итераций, сколько элементов есть в массиве.

```csharp
int[] myArray = { 10, 2, 4, 75 };  
for (int i = 0; i < myArray.Length; i++)  
{  
		Console.WriteLine(myArray[i]);
}
```

Переменную `i` мы можем использовать как индекс массива и перебрать переменные от 0 и до последней, пока `i` будет меньше длины массива
