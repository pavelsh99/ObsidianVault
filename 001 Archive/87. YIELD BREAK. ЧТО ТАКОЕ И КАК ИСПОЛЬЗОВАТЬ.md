# `yield break` как работает

`yield break` так же как `yield return` разворачивается в state-машину.

Работает он по аналогии с`break` в цикле, но цикла нет, а внутри state-машина.

`yield break` прерывает итерацию в случае, когда нам нужно прервать итерацию без возвращения какого- либо значения

# Пример работы

```csharp
        public static IEnumerable<int> GetNumbers()
        {
            yield return 1;
            yield return 2;
            yield break;
            yield return 3;
        }
        static void Main(string[] args)
        {
            foreach (var number in GetNumbers())
            {
                Console.WriteLine(number);
            } 
        }
```

У нас есть `IEnumerable`, который по одному возвращает числа.

# Пример 2

```csharp
        public static IEnumerable<int> SortByNumbers(IEnumerable<int> Numbers)
        {
            foreach (int number in Numbers)
            {
                if (number % 2 == 0)
                    yield return number;
            }
        }
        static void Main(string[] args)
        {
            int[] MyNumbers = { 1, 2, 3, 4, 5, 6, 7, 8 };

            foreach (int number in SortByNumbers(MyNumbers))
            {
                Console.WriteLine(number);
            }
        }
```


У нас есть сортировка четных чисел.
![[Pasted image 20260305110827.png]]

Но что если туда попадет `Null`?

Мы получим исключение, поэтому, чтобы не допустить этого, поможет `yield break`

```csharp
        public static IEnumerable<int> SortByNumbers(IEnumerable<int> Numbers)
        {
	        if (Numbers == null) // ТУТ
		        yield break;
		    
            foreach (int number in Numbers)
            {
                if (number % 2 == 0)
                    yield return number;
            }
        }
        static void Main(string[] args)
        {
            int[] MyNumbers = { 1, 2, 3, 4, 5, 6, 7, 8 };

            foreach (int number in SortByNumbers(MyNumbers))
            {
                Console.WriteLine(number);
            }
        }
```
