Ключевое слово `break` прерывает выполнение любого цикла `while`, `do while` ,`for` , `foreach`.

```csharp
    for (int i = 0; i < 1000; i++)
    {
        Console.WriteLine(i);

        if (i == 100)//консоль прочитает числа до ста
            break;
    }
```

Другой пример:

```csharp
            for (int i = 0; i < 100 ; i++)
            {
                string msg = Console.ReadLine();    


                if (msg == "stop")
                {
                    break;
                }
                Console.WriteLine(i);
            }
```

Программа будет выполнять 100 итераций каждый раз ожидая текст, но если она увидит слово stop то прекратит выполнение.
