# ?? - оператор объединения

Оператор объединения с `null` это синтаксический сахар. 

Он позволяет удобно присвоить дефолтное значение для ссылочных классов.

В такой ситуации мы получим ошибку или не получим ничего:

```csharp
            string str = null;

            Console.WriteLine(str);
```

чтобы обработать такую ситуацию мы можем использовать довольно много кода 

```csharp
            string str = null;

            if (str == null)
            {
                Console.WriteLine("нет данных");
            }
            else
            Console.WriteLine(str);
```

Для таких ситуаций и придуман оператор `null` объединения

```csharp
string str = null;
Console.WriteLine(str ?? "нет данных");  // Выведет: нет данных

str = "Привет";
Console.WriteLine(str ?? "нет данных");  // Выведет: Привет
```
