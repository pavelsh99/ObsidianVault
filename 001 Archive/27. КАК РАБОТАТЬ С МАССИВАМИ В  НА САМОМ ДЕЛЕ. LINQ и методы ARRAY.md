Тут будут типичные решения, которые сделают сложные вещи легкими.

Одним из таких инструментов является **LINQ**.

Массивы наследуют класс **Array**, поэтому можно использовать статистические методы этого класса.

Всё это получается при использовании `using System.Linq`

Найти наибольший элемент массива:
```csharp
            int[] myArray = { 1, 2, 3, 4, 5, 6, 7, 8, };

            int result = myArray.Max();
```

Найти наименьший элемент массива:
```csharp
            int[] myArray = { 1, 2, 3, 4, 5, 6, 7, 8, };

            int result = myArray.Min();
```

Сумма всех элементов:
```csharp

            int[] myArray = { 1, 2, 3, 4, 5, 6, 7, 8, };

            int result = myArray.Sum();
```

Сумма всех четных элементов:
```csharp
            int[] myArray = { 1, 2, 3, 4, 5, 6, 7, 8, };

            int result = myArray.Where(i => i % 2 ==0).Sum(); //определение
				            //тут
```
