Условный оператор `switch`  выполняет то же самое что `if` `else` и используется для  разветвления кода.

```csharp
            switch (switch_on) 
            {

            }
```

В `switch` передается не `true` `false` как в `if` `else`, а может быть например значение:

```csharp
            int a = int.Parse(Console.ReadLine());

            switch (a)
            {
                case 1://двоеточие // тут указывается тип данных.
                    Console.WriteLine("Вы ввели число 1");
                    break;//точка с запятой

                case 2:
                    Console.WriteLine("Вы ввели чисо 2");
                    break;
                default:
	                Console.WriteLine("Я не знаю что это за число")
	                break;
            }
```

Можно использовать фигурные скобки для работы с `switch`:

```csharp
            switch (a)
            {
                case 1:
                    {//вот они
                        Console.WriteLine("Вы ввели число 1");
                        break;
                    }//вот они
```

Можно объединить несколько кейсов, для получения одного результата:

```csharp
            switch (a)
            {
                case 1:
                case 2:
                            Console.WriteLine("Вы ввели число 1 или 2");
                            break; 
            }
```

Можно задействовать `string` для использования `switch`:

```csharp
            string input = Console.ReadLine();  

            switch (input)
            {
                case "+":
                    Console.WriteLine("вы ввели +.");
                    break;

                case "-":
                            Console.WriteLine("Вы ввели -.");
                            break; 
                        
                default:
                    Console.WriteLine("Я не знаю что это");
                    break;
            }
```
