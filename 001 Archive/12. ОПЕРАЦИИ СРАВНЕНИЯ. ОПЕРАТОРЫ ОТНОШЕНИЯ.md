# Операторы отношений

Операции сравнения - это действия, которые выполняются операторами отношений.

```csharp
       a == //b равно?
       a != //b не равно?
       a > //b больше?
       a < //b меньше?
       a >= //b больше или равно?
       a <= //b меньше или равно?
        
```

Результатом операторов отношений будет `true` или `false`, которые можно хранить в переменную `bool`.

**Важно!** для проверки на равенство нужно использовать `==`, а не `=`:

```csharp

    int a = 5;
 
    int b = 9;

    Console.WriteLine(a == b); // проверка на неравенство
  
     // Консоль покажет False 
```
