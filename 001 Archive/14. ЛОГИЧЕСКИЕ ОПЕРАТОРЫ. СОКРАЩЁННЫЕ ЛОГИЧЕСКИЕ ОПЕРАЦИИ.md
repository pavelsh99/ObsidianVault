# `&&`                        **Сокращенное И**
# `||`                        **Сокращенное ИЛИ**
# `&`                          **И**
# `|`                          ИЛИ
# `!`                          **НЕ (унарный)**

# Использование унарного оператора `!`НЕ.

```csharp
            bool Isinfected = true;

            if (!Isinfected)// Персонаж НЕ Isinfected.
            {
                Console.WriteLine("Персонаж не заражен");

            }
            else
            {
                Console.WriteLine("Персонаж заражен");
            }
```

# Использование бинарных И и ИЛИ

`&&` используется, когда нужно проверить наличие нескольких переменных:

```csharp
            bool HasNoCooling = true;

            bool isHighTemperature= true;

            if (isHighTemperature && HasNoCooling)// 2 Параметра
            {
                Console.WriteLine("УГРОЗА ПОВРЕЖДЕНИЯ ПРОЦЕССОРА");
            }
```

Только если 2 параметра совпадут, консоль выведет сообщение

## Использование оператора`||` (ИЛИ):

Тут необходимо одно из двух условий, для вывода сообщения.

```csharp
            bool HasNoCooling = false;

            bool isHighTemperature = true;

            if (isHighTemperature || HasNoCooling) //Если или то или то
            {
                Console.WriteLine("УГРОЗА ПОВРЕЖДЕНИЯ ПРОЦЕССОРА");
            }
```

`||`с введением дополнительной переменной

```csharp
            int FanSpeed = 3000;

            bool HasNoCooling = FanSpeed <= 0;

            bool isHighTemperature = true;

            if (isHighTemperature || HasNoCooling)
            {
                Console.WriteLine("УГРОЗА ПОВРЕЖДЕНИЯ ПРОЦЕССОРА");
            }
```

## Отличие `|` и `&` от `||` и `&&`

==При их использовании не используется часть выражения, если она не нужна. А | и & проверяют все условия==

```csharp
            bool HasNoCooling = false;

            bool isHighTemperature = true;

            if (isHighTemperature & HasNoCooling)
            {
                Console.WriteLine("УГРОЗА ПОВРЕЖДЕНИЯ ПРОЦЕССОРА");
            }
```

```csharp
            bool HasNoCooling = true;

            bool isHighTemperature = false;

            if (isHighTemperature | HasNoCooling)
            {
                Console.WriteLine("УГРОЗА ПОВРЕЖДЕНИЯ ПРОЦЕССОРА");
            }
```
В такой ситуации `&` будет проверено 2 параметра , не смотря на то, что первый  не подходит, в то время, как `&&` уже увидев первый параметр, выдаст предупреждение.

>"Таким образом `||` и `&&` быстрее**"
