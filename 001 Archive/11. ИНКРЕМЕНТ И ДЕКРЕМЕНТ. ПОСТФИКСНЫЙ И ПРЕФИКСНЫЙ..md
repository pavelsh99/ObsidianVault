- *Инкремент и декремент* это унарные операции, они имеют ==постфиксную== и ==префиксную== форму

- **Суть операций** - это увеличение или уменьшение значения на единицу

# Инкремент:

```csharp
         int a = 1;

         a++; // a+1

         Console.WriteLine(a); // a = 2
```

# Декремент:

```csharp
         int a = 1;

         a--; // = a-1 

        Console.WriteLine(a); // a = 0
        
```

- Эта форма записи удобной и используется в различных счетчиках.

- Она также унарная, потому что оператор инкремента потребляет один операнд.

- Существует **постфиксная** и **префиксная** форма записи

# Префиксная форма инкремента и декремента:

```csharp
    ++a 
    
    --a
```

# Операция постфиксного инкремента имеет наименьший приоритет:

```csharp

        int a = 1;

        Console.WriteLine(a++);// НА КОНСОЛЬ ВЫВЕДЕТСЯ 1
        
```

При такой записи a будет равно ==одному==, потому что сначала выводится на консоль, а только потом  увеличивается. Это относится и к декременту.

# Операция префиксного инкремента имеет наивысший приоритет:

```csharp

        int a = 1;

        Console.WriteLine(++a);// НА КОНСОЛЬ ВЫВЕДЕТСЯ 2
        
```

На консоли будет число 2, потому что сначала будет присвоено значение, а только потом произойдет вывод на консоль. Это относится и к декременту.

>Префиксный инкремент имеет наивысший приоритет:

```csharp

        int a = 2;

        a = ++a * a; // (2+1)x3 = 9 (

        Console.WriteLine(a); // a = 9
        
```

Сначала инкремент присваивает `a = 2 + 1` , а потом происходит умножение на `a` то есть на 3 , (`3 * 3 = 9`)

```csharp

            int a = 2;

            a = a++ * a; // (2+1) x 2 = 6

            Console.WriteLine(a); // a = 6
            
```

тут при умножении `a = 2`, потому что постфиксный инкремент имеет низший приоритет , поэтому получается `(2 + 1) * 2`.
