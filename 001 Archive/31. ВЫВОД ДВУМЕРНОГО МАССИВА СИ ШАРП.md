Для вывода массива на консоль нужно иметь доступ ко всем его элементам.

Делать это нужно так, чтобы при изменении размера массива логика работала.

# Вывод двумерного массива с помощью foreach

```csharp
            int[,] myArray = new int[,]
            {
                {12,51,43,19,14},
                {44,24,63,18,76},
                {15,98,22,17,53}
            };
            foreach (var item in myArray)
            {
                Console.WriteLine(item);
            }
```

# Вывод двумерного массива таблицей c  вложенным `for`

```csharp
            int[,] myArray = new int[,]
            {
                {12,51,43,19,14},
                {44,24,63,18,76},
                {15,98,22,17,53}
            };
Console.WriteLine(myArray.Rank);//Rank позволяет понять колличество измерений
```

Чтобы узнать количество элементов в определенном измерении используется `.Getlenght`

```csharp
Console.WriteLine(myArray.GetLength(0));//GetLengh(<измерение>)
```

>Нумерация измерений начинается с 0!

```csharp

            int[,] myArray = new int[,]
            {
                {12,51,43,19,14},
                {44,24,63,18,76},
                {15,98,22,17,53}
            };

            int height = myArray.GetLength(0);
            int width = myArray.GetLength(1);


            for (int y = 0; y < height; y++)//двигается по y
            {
                for (int x = 0; x < width; x++)//двигается по x
                {
                    Console.Write(myArray[y, x] + "\t");
                }
                Console.WriteLine();//пропускает строку
```
