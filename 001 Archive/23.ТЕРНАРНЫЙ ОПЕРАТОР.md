###### Тернарный оператор - это оператор который выполняет действия над 3 операндами.

>Записывается он так:
>`[первый операнд - условие] ? [второй операнд if_true] : [третий опреранд if_false]`

**Он делает то же самое, что `if` `else`, но он компактнее**

Например есть код, с `if else`, который проверяет пароль.

```csharp
bool accessAllowed;  
  
string StoredPassword = "1234";  
string enteredPassword = Console.ReadLine();  
  
if (enteredPassword == StoredPassword)  
{  
    accessAllowed = true;  
}  
else  
{  
    accessAllowed = false;  
}  
Console.WriteLine(accessAllowed);
Console.ReadLine();
```

Тернарным оператором это можно записать так:

```csharp
accessAllowed = enteredPassword == StoredPassword ? true : false;
```

Другой пример использования : код который выводит числа больше нуля, а меньше нуля приравнивает к нулю.

```csharp
int input = int.Parse(Console.ReadLine());

int outputData = input < 0 ? 0 : input; //"`если условие? тогда : иначе`"

Console.Writeline(input);
```
