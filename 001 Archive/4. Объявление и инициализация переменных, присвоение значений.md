 ### **Объявление** = создание переменной:
 
```Csharp
 int a; //тип_данных название_переменной
```
 
 ### **Инициализация переменных** - это присвоение значения переменной в первый раз.
 
```csharp
  int c;
  c = 33;//"=" это оператор присваивания 
```

### **Создание** и **инициализация** может быть создана и инициализирована в один момент:

```csharp
int a = 9;
```

### Можно объявлять переменные **группой**:

```csharp

int a,b,c;
int a = 2;
int b = 5;
int c = 9;

```
### Переменные Char объявляются так:

```csharp

char a = 'A';

```

### Переменные **string** объявляются так:

```csharp

string a = "Hello";

```
