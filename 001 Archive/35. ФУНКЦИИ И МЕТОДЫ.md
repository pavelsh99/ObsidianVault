Методы упрощают код, позволяют обращаться к нужным фрагментам кода, который находится внутри тела метода.

Синтаксис метода: 
```csharp
модификаторы тип_возвращаемого значения название_метода(параметры)
{
тело метода
}
```

Модификаторы влияют на то как ведут себя методы

Тип возвращаемого значения это тип данных для результата выполнения метода.

Название мы придумываем сами, может быть любое.

Параметры это не обязательная часть метода, это **значения, которые мы передаём методу**, чтобы он мог с ними работать. Это как **входные данные** для метода.
```csharp
        static int Sum (int a, int b) // не обязательно называть a b
        {
            int result = a + b;

            return result;   // можно return a + b;
        }
        static void Main(string[] args)
        {
            int a = int.Parse(Console.ReadLine());
            int b = int.Parse(Console.ReadLine());
            int c  = Sum (a, b);
        }
```

Существует ключевое слово `void` , которое значит, что метод ничего не возвращает.
```csharp
void SayHello()
{
    Console.WriteLine("Привет!");
}
```

В c# можно вызвать методы раньше их объявления в коде.
